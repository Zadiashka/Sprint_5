Автоматизация тестирования — Sprint 5  
Полностью автоматизированный набор UI‑тестов для веб‑приложения **Stellar Burgers**  
(https://stellarburgers.education-services.ru)

Проект выполнен в рамках Sprint 5 курса по автоматизации тестирования.  
Все тесты написаны на **Python + Selenium + Pytest**.

---

Что сделано

Полная автоматизация ключевых пользовательских сценариев:
- Регистрация нового пользователя  
- Авторизация (вход) разными способами  
- Выход из аккаунта  
- Навигация по разделам сайта  
- Работа вкладок конструктора  
- Переходы в личный кабинет и обратно  
- Проверка поведения SPA‑навигации (React Router)

Исправлены и стабилизированы:
- Локаторы под новый DOM сайта  
- Логика переходов `/account → /login → /`  
- Ожидания элементов (WebDriverWait)  
- Обработка React‑задержек  
- Клики через безопасную функцию `safe_click`  
- Генерация уникальных email для регистрации  

Удалены:
- Старые неактуальные локаторы  
- Скриншоты и HTML‑дампы прошлых падений  
- Временные файлы и черновики  

---

Какие тесты реализованы

1. Регистрация и логин  
Файл: `tests/test_registration_and_login.py`

Проверяется:
- успешная регистрация  
- логин через форму  
- логин через кнопку «Личный кабинет»  
- логин через кнопку «Войти» на главной  
- логин через кнопку в форме регистрации  
- логин через кнопку восстановления пароля  

---

2. Выход из аккаунта  
Файл: `tests/test_logout.py`

Проверяется:
- корректный выход через кнопку «Выход»  
- редирект на страницу логина  
- невозможность вернуться в аккаунт без повторного входа  

---

3. Навигация профиля  
Файл: `tests/test_profile_navigation.py`

Проверяется:
- переход в личный кабинет  
- логин → возврат на главную  
- повторный вход в личный кабинет  
- отображение активной вкладки «Профиль»  
- переход в конструктор  
- переход по логотипу  

---

4. Вкладки конструктора  
Файл: `tests/test_constructor_tabs.py`

Проверяется:
- корректная работа вкладок «Булки», «Соусы», «Начинки»  
- отображение соответствующих элементов  

---

5. Методы логина  
Файл: `tests/test_login_methods.py`

Проверяется:
- вход через разные кнопки и ссылки  
- корректные редиректы

Как запустить тесты

Установка зависимостей:

```bash
pip install -r requirements.txt

Запуск всех тестов:
pytest -q

Запуск конкретного файла:
pytest tests/test_profile_navigation.py -q
